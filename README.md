# GRIP Insight

Interactive research reading portfolio for the paper **Retrieval as Generation: A Unified Framework with Self-Triggered Information Planning**.

## Tech Stack

- Next.js App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Recharts
- Lucide React

## Local Development

```bash
npm install
npm run dev
```

Open <http://127.0.0.1:3000>.

## Validation

```bash
npm run lint
npm run build
```
